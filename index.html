Skill Saya

HTML adalah bahasa markup standar yang digunakan untuk membuat halaman web.

CSS adalah bahasa yang digunakan untuk mendesain dan mengatur tampilan halaman web, seperti warna, font, dan layout.

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web interaktif, memungkinkan manipulasi elemen HTML, pengelolaan event, dan komunikasi dengan server secara dinamis..
